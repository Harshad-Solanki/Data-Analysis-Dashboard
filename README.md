# Hospital Emergency Room Dashboard
## 📊 Project Overview
This project focuses on analyzing hospital emergency room data using Microsoft Excel to generate meaningful insights and support better decision-making in healthcare operations.  The dashboard was designed to monitor patient flow, satisfaction levels, department referrals, and attendance trends through interactive visualizations and KPI tracking.

## 🔑 Key Features
Cleaned and transformed raw hospital data for analysis
Built interactive KPIs:
Total Patients
Average Wait Time
Patient Satisfaction Score
Created dynamic Pivot Charts and Slicers for Month-wise and Year-wise analysis
Analyzed patient admission status and attendance performance
Performed gender-wise and age-group analysis
Visualized department referral trends
Designed a clean and professional dashboard for easy decision-making
